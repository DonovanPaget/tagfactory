This program will take a csv named "pricebook.csv" and create tags based off
of the information in that csv file.

The formatting of the csv file MUST BE:

BRAND NAME | ITEM NAME | PRICE

where BRAND NAME must exactly equal one of the following:
atomic burton full tilt head jones k2 line look marker ride rossignol salomon tyrolia

there must be no empty lines or header lines.

pricebook.csv should be moved into the resources folder prior to running the program.

the tag will be 1200 pixels wide x 555 pixels tall

the tags are saved in tags.
sheets of tags are saved in tags/sheets

TAG FACTORY WILL ONLY MAKE SHORT WIDE TAGS (for boots)
